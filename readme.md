# DLDRPSY AI 智能助手

一个适配大连东软信息学院心理测评页面的 Tampermonkey 脚本，提供可视化控制面板与 AI 答题能力。

## 功能特性

- 可拖拽、可最小化的控制面板
- AI 配置保存（API Key / Endpoint / Model）
- 单题 AI 解答（当前题自动选择）
- 全自动 AI 答题（逐题作答，支持手动停止）
- 复制当前题目与选项
- 上一题 / 下一题测试按钮
- 自动识别页面“自动翻页”行为，减少跳题

## 运行环境

- 浏览器：Chrome / Edge
- 扩展：Tampermonkey
- 站点：`https://dldrpsy.neusoft.edu.cn/*`

## 安装方式

1. 安装 Tampermonkey 扩展。
2. 新建用户脚本。
3. 将 `tampermonkey-dldrpsy-inspector.user.js` 全部内容粘贴进去并保存。
4. 打开 DLDRPSY 测评答题页面，右侧会出现控制面板。

## 使用说明

### 1. 配置 AI

在面板里展开 `AI 配置`，填写：

- API Key
- API Endpoint
- Model

点击 `保存 AI 配置`。

### 2. 单题解答

点击 `AI 解答当前题目`：

- 脚本读取当前题干与选项
- 请求 AI 返回答案字母
- 自动点击对应选项

### 3. 全自动答题

点击 `开始全自动 AI 答题`：

- 每题调用 AI 作答
- 自动处理“页面自动翻题”或“手动下一题”
- 到最后一题且已作答后自动停止
- 再次点击按钮可手动停止

## 日志说明

面板底部日志区会显示运行状态，例如：

- `正在请求 AI（题号 X）`
- `题号 X 已选: A`
- `已到最后一题，自动答题停止`
- `AI 解题失败: ...`

## 常见问题

### 1. 提示“请先配置 API Key”

未保存 API 配置。请先填写并保存 API Key。

### 2. 题目没有被选中

可能是页面结构临时变化或网络波动导致。可先使用单题按钮验证，再开启全自动。

### 3. 自动答题提前停止

通常由以下原因触发：

- 当前题未成功作答
- 页面未检测到已选项
- 到达最后一题并已作答

查看日志可快速定位问题点。
